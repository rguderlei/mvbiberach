# Website MV Biberach

## Getting started

- install Hugo https://gohugo.io/
- clone this repo
- run ```hugo server```

## Tips and tricks

build background image for carousel

```
montage grid/*.jpg -tile 8x5 -geometry +0+0 -background none grid.jpg
```
